# Containers

Los contenedores son un elemento fundamental de construcción de Bootstrap que contienen, rellenan y alinean tu contenido dentro de un dispositivo o viewport dado.

## Cómo funcionan

Los contenedores son el elemento de layout más básico en Bootstrap y son **requeridos cuando usas nuestro sistema de grid por defecto**. Los contenedores se usan para contener, rellenar y (a veces) centrar el contenido dentro de ellos. Aunque los contenedores _pueden_ estar anidados, la mayoría de layouts no requieren un contenedor anidado.

Bootstrap viene con tres contenedores diferentes:

- `.container`, que establece un `max-width` en cada breakpoint responsivo
- `.container-{breakpoint}`, que es `width: 100%` hasta que se alcanza el breakpoint especificado
- `.container-fluid`, que es `width: 100%` en todos los breakpoints

## Contenedor por defecto

Nuestra clase `.container` por defecto es un contenedor responsivo de ancho fijo, lo que significa que su `max-width` cambia en cada breakpoint.

```html
<div class="container">
  <!-- Contenido aquí -->
</div>
```

## Contenedores responsivos

Los contenedores responsivos te permiten especificar una clase que es 100% de ancho hasta que se alcanza el breakpoint especificado, después del cual aplicamos `max-width`s para cada uno de los breakpoints más altos. Por ejemplo, `.container-sm` es 100% de ancho para empezar hasta que se alcanza el breakpoint `sm`, donde escalará con `md`, `lg`, `xl`, y `xxl`.

```html
<div class="container-sm">100% de ancho hasta breakpoint small</div>
<div class="container-md">100% de ancho hasta breakpoint medium</div>
<div class="container-lg">100% de ancho hasta breakpoint large</div>
<div class="container-xl">100% de ancho hasta breakpoint extra large</div>
<div class="container-xxl">100% de ancho hasta breakpoint extra extra large</div>
```

## Contenedores fluidos

Usa `.container-fluid` para un contenedor de ancho completo, que abarca todo el ancho del viewport.

```html
<div class="container-fluid">
  ...
</div>
```

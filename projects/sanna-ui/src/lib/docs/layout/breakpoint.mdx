# Breakpoints

Los breakpoints son anchos personalizables que determinan cómo se comporta tu diseño responsivo en diferentes tamaños de dispositivos o viewports en Bootstrap.

## Conceptos fundamentales

- Los breakpoints son los bloques de construcción del diseño responsivo. Úsalos para controlar cuándo tu layout puede adaptarse a un viewport o tamaño de dispositivo particular.
- Usa media queries para arquitecturar tu CSS por breakpoint. Las media queries son una característica de CSS que te permite aplicar estilos condicionalmente basado en un conjunto de parámetros del navegador y sistema operativo. Más comúnmente usamos **min-width** en nuestras media queries.
- Mobile first, diseño responsivo es el objetivo. El CSS de Bootstrap apunta a aplicar el mínimo de estilos para hacer que un layout funcione en el breakpoint más pequeño, y luego agrega capas de estilos para ajustar ese diseño para dispositivos más grandes. Esto optimiza tu CSS, mejora el tiempo de renderizado, y proporciona una gran experiencia para tus visitantes.

## Breakpoints disponibles

Bootstrap incluye seis breakpoints por defecto, a veces referidos como grid tiers, para construir de manera responsiva. Estos breakpoints pueden ser personalizados si estás usando nuestros archivos Sass fuente.

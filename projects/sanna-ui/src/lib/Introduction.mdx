# 🎨 Sanna UI - Documentación de Componentes 🚀


Explora nuestros componentes navegando por las categorías en el panel lateral:

### 📋 **Componentes Disponibles**

- **🔘 Button** - Botones interactivos con múltiples variantes y estilos
- **📝 Text** - Componentes de texto con tipografía personalizada  
- **📊 Table** - Tablas responsive con paginación y ordenamiento
- **💬 MessageBox** - Cajas de mensajes para notificaciones y alertas
- **🏷️ Heading** - Títulos y encabezados con diferentes tamaños
- **🎯 Icon** - Iconos de FontAwesome integrados (150+ disponibles)
- **📄 Input** - Campos de entrada con validación

### ✨ **Características Principales**

- **🎨 Diseño Moderno**: Colores personalizados con verde principal `#5BAB5F`
- **📱 Totalmente Responsive**: Adaptable a todos los dispositivos
- **🔧 Altamente Configurable**: Múltiples props y opciones de personalización
- **⚡ Fácil de Usar**: Sintaxis clara y documentación completa
- **🔤 Tipografía Premium**: Plus Jakarta Sans integrada automáticamente
- **🎯 FontAwesome Incluido**: Más de 150 iconos listos para usar

### 🛠️ **Sintaxis Recomendada**

Nuestros componentes soportan sintaxis flexible:

```html
<!-- ✅ Valores estáticos (recomendado) -->
<sa-button label="Guardar" variant="primary" size="medium"></sa-button>
<sa-icon name="heart" color="#ff0000" size="lg"></sa-icon>

<!-- ✅ Valores dinámicos -->
<sa-button [label]="buttonText" [variant]="buttonType"></sa-button>
<sa-icon [name]="iconName" [color]="iconColor"></sa-icon>

<!-- ✅ Sintaxis mixta -->
<sa-messagebox [message]="notification" type="success" iconName="check"></sa-messagebox>
```
